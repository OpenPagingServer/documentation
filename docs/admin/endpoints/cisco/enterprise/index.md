# Cisco Enterprise Phones

This section applies to the following phones running UCM/Enterprise Firmware
- 9800 series
- 8800 series
- 7800 series
And all models of the following phones
- 7900 series
- 8900 series
- 9900 series
- 6900 series
- 3900 series

Visual and audio messages can be sent to Cisco UCM phones. Delivery of audio requires the Cisco IP Phones module to be installed, because they require proprietary CGI execute commands.
