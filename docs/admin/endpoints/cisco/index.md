# Cisco IP Phones

Cisco IP Phones, when connected to Open Paging Server using the offical module, can be used to receive broadcasts. Audio alongside a visual notification are sent.

Enterprise/SEP, SPA, 3PCC/MPP, and phoneOS phones are supported.

Sending messages to phones rather than just overhead speakers

Open Paging Server was desgined for Cisco IP Phones from the ground up.
